#### 002. Git Convention

> 일관적인 commit message를 남기는 습관을 들이기 위한 규칙입니다.

<br>

| #    | type | description           |
| ---- | ---- | --------------------- |
| 1    | baek | baekjoon 문제 풀이    |
| 2    | prog | programmers 문제 풀이 |
| 3    | swea | swea 문제 풀이        |
| 4    | docs | 문서 작업             |

<br>

```bash
# example

$ git commit -m 'baek: "1240 노드사이의 거리"'
```
